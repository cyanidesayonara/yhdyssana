# yhdyssana

A Vue app that checks whether or not user input words form a proper compound word in Finnish.

https://yhdyssana.herokuapp.com/

## Create .env file

```
PORT=PORT
MONGODB_URI=MONGODB_URI
```

## Upload words to database

```
python wordUploader.py
```

## Project setup

```
npm install
```
